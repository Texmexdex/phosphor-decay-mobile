# Phosphor Decay Mobile

Mobile-optimized version of Phosphor Decay - An audio-visual performance instrument for touch devices.

## Mobile Features

### Touch Gestures
- **1 Finger Drag**: Pan the feedback canvas
- **2 Finger Pinch**: Zoom in/out
- **2 Finger Rotate**: Rotate the feedback canvas

## Performance Tips

- Close other apps for better performance
- Use landscape mode for wider canvas
- Reduce FPS if experiencing lag
- Lower pixelate/effect values on older devices
