## Features

### Visual System
- **Analog Video Feedback** - Canvas captures itself with natural latency
- **3D Rotating Shapes** - Wireframe polyhedra spinning in perspective
- **Sacred Geometry** - Merkaba, Metatron's Cube, Fibonacci spirals
- **Drawing System** - Record, loop, and layer your gestures
- **Kaleidoscope Mode** - Mirror segments creating infinite symmetry
- **Color Chase** - Colors propagate and flow through the feedback
- **Continuous Motion** - Pan/zoom/rotate velocity with wrapping
- **Mouse Control** - Direct manipulation of the feedback space

### Audio System
- **Visual Sequencer** - Music generated from fractal motion detection
- **18+ Synth Presets** - FM, analog, digital, metallic, organic
- **Timbre Control** - Filters, envelopes, effects per instrument
- **Music Theory Engine** - Chord progressions, scales, harmonic structure
- **Rhythm Templates** - Ambient to drum & bass
- **Real-time Reactivity** - Audio follows visual motion and brightness

### The Loop
- High-DPI rendering for pristine feedback quality
- Adjustable feedback parameters (zoom, rotation, pan)
- Visual effects (invert, contrast, hue shift, pixelation)
- Audio effects (reverb, delay, distortion, bitcrush)
- Fullscreen performance mode
- UI hide (press 'U') for pure immersion

## Quick Start

```bash
# No build required - pure vanilla JavaScript
python -m http.server 8000

# Open: http://localhost:8000/video-synth/
```

1. Click **INITIALIZE_SYSTEM**
2. The feedback loop begins automatically
3. Add shapes, draw lines, adjust parameters
4. Watch the singularity unfold
5. Listen to it sing itself into existence
